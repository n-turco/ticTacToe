# Tic Tac Toe

## Project Overview

Tic Tac Toe is a WPF-based implementation of the classic Tic Tac Toe game, focusing on logic-based programming and user interaction. This project demonstrates game state management, win/loss condition detection, and responsive GUI design.

## Key Features

- Two-player game mechanics
- Automatic win/loss condition detection
- Player turn transitions
- Game reset functionality
- Visually polished WPF interface
- Real-time game state updates

## Technical Stack

- C#
- WPF (Windows Presentation Foundation)
- .NET Framework

## How to Run

1. Ensure Visual Studio and .NET Framework are installed
2. Open the project in Visual Studio
3. Build the solution:

   ```
   dotnet build
   ```

4. Run the application:

   ```
   dotnet run
   ```

5. The game will launch and you can begin playing
